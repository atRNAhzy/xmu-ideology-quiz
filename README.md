# XMU Ideology Quiz

厦门大学思政课选择题复习软件（Windows）。

用于思政类课程的选择题刷题与复习，支持单选 / 多选题。
目前包含题库：
- 马原 2025–2026 学年第一学期
- 马原在线题库版
- 毛概在线题库版
- 史纲在线题库版
- 习概 2024–2025 学年第二学期

题库文件可直接通过软件界面的下拉栏进行选择与加载。

在线题库来源：https://f5soft.github.io/RubbishMaoGai/


## 下载

Release v1.0.0（Windows 可执行程序与源码）  
https://github.com/atRNAhzy/xmu-ideology-quiz/releases/tag/v1.0.0

## 功能

- 选择题刷题：单选 / 多选
- 顺序练习 / 随机练习
- 题库下拉选择
- 本地运行，无需联网


## 使用方法（Windows）

1. 下载 [`XMU-Ideology-Quiz_v1.0.0.zip`](https://github.com/atRNAhzy/xmu-ideology-quiz/releases/tag/v1.0.0)
2. 解压后双击运行 `XMU-Ideology-Quiz_v1.0.0.exe`
3. 如被 SmartScreen 拦截：更多信息 → 仍要运行
4. 选择题库开始刷题
